ordered list

An ordered list starts with the ol opening tag and ends with /ol closing tag. Each list item starts with the li tag. The list items are usually marked with numbers letters or roman numbers.

example of ordered list

  1. mangos
  2. oranges
  3. bananas




unordered list

An unordered list starts with the ul opening tag and ends with /ul closing tag. Each list item starts with the li tag. The list items are usually bulleted (small circle).

example of an unordered list

  • mangos
  • oranges
  • bananas




description list

A description list is a list of terms, with a description of each term.The dl tag defines the description list, the dt tag defines the term (name), and the dd tag describes each term:

example of a description list

Coffee
- black hot drink
Milk
- white cold drink
